Bilan de mon BTS SIO SLAM

Rétrospective sur deux années de formation enrichissantes

Ce que j'ai appris

Compétences techniques

JavaScript SQL Java Python Git HTML/CSS Framework MVC API REST

Compétences transversales

  • Gestion de projet
  • Travail en équipe et collaboration
  • Communication technique et documentation
  • Analyse et résolution de problèmes
  • Veille technologique

Ce que j'ai apprécié

Points positifs

  • La diversité des projets pratiques
  • L'alternance théorie/pratique
  • L'apprentissage par la pratique

Points à améliorer

  • Temps limité pour les projets personnels

Ce que je ferais différemment

Mes recommandations

  • Commencer la veille technologique plus tôt pour mieux suivre les évolutions
  • Participer à plus de projets open source pour gagner en expérience
  • Développer davantage de projets personnels en parallèle des cours
  • Créer un réseau professionnel plus tôt dans la formation
  • Approfondir certaines technologies spécifiques en autonomie

En conclusion

Ces deux années de BTS SIO SLAM ont été fondamentales dans ma formation professionnelle. Malgré quelques points d'amélioration possibles, j'ai acquis une base solide de compétences techniques. Cette formation m'a permis de confirmer mon choix de carrière dans le développement informatique et m'a donné les outils nécessaires pour continuer à évoluer dans ce domaine.
