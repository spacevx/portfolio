Politique de Confidentialité

Protection de vos données personnelles

Introduction

La présente politique de confidentialité précise qu'aucune donnée personnelle n'est collectée par moi-même, Ahcene Zidahnal, lors de votre navigation sur ce portfolio.

Ce site étant hébergé sur GitHub Pages, veuillez noter que GitHub en tant qu'hébergeur peut collecter certaines informations techniques. Pour en savoir plus sur les données que GitHub peut collecter, je vous invite à consulter leur politique de confidentialité.

Cette politique peut être modifiée ou complétée à tout moment, notamment pour se conformer à toute évolution réglementaire.

Absence de collecte personnelle

En tant que propriétaire de ce portfolio, je n'ai mis en place aucun mécanisme de collecte de données personnelles :

  • Je ne collecte aucune donnée de connexion
  • Je n'enregistre aucune donnée de navigation
  • Je n'utilise aucun cookie
  • Je n'ai implémenté aucun système de suivi ou d'analyse

Ce site est un simple portfolio statique hébergé sur GitHub Pages, sans base de données ni traitement côté serveur de ma part.

Absence d'utilisation de données

Puisque ce site ne collecte aucune donnée personnelle :

  • Aucune donnée n'est utilisée pour suivre votre navigation
  • Aucune analyse de comportement n'est effectuée
  • Aucune statistique personnalisée n'est établie
  • Les éventuels contacts se font uniquement via l'adresse e-mail indiquée, sans stockage automatique d'informations

Données collectées par l'hébergeur

Ce site est hébergé par GitHub Pages, un service d'hébergement de sites web statiques fourni par GitHub, Inc.

Bien que je ne collecte personnellement aucune donnée, GitHub en tant qu'hébergeur peut collecter certaines informations techniques comme :

  • Des adresses IP
  • Des données de trafic
  • Des informations sur le navigateur utilisé
  • Des statistiques d'accès aux pages

Ces collectes sont soumises à la politique de confidentialité de GitHub. Pour en savoir plus, vous pouvez consulter leur politique de confidentialité sur le site officiel de GitHub.

Respect de la vie privée

Je respecte votre vie privée, c'est pourquoi ce portfolio est conçu pour être le moins intrusif possible :

  • Aucune information personnelle n'est requise pour naviguer sur le site
  • Aucun formulaire de collecte d'informations n'est présent
  • Aucun traitement automatisé de vos données n'est effectué
  • Aucun partage de données avec des tiers n'est possible puisqu'aucune donnée n'est collectée

Cette approche est conforme aux principes du RGPD et de la loi Informatique et Libertés qui visent à minimiser la collecte des données personnelles.

Cookies

Mon site n'utilise aucun cookie. En tant que site statique hébergé sur GitHub Pages, aucun mécanisme de suivi ou de stockage local n'est mis en place.

Vous n'avez donc pas besoin de vous soucier de la gestion des cookies lors de votre navigation sur ce portfolio.

Contact et mentions légales

Pour toute question relative à cette politique de confidentialité ou pour obtenir plus d'informations sur les aspects liés à la protection des données, vous pouvez me contacter directement :

  • Par email : ahcene-zidahnal@gmail.com

Je vous rappelle que les échanges par email sont traités manuellement et qu'aucun système automatisé de traitement des données n'est utilisé de ma part pour gérer ces communications.

Pour consulter les mentions légales complètes de ce site, vous pouvez vous référer à la page Mentions légales.
